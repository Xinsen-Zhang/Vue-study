## 如何使用 Vue
### 开发模式
`Vue.js`是一个渐进式的 JavaScript 框架, 根据项目需要, 你可以选择从不同的维度使用它
* 如果只是想要体验`Vue.js`带来的快感, 或者开发几个简单的 HTML5页面或者小应用, 可以直接使用 script 加载 CDN 文件.(e.g. 数据循环)
